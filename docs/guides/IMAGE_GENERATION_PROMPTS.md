## üìù ISTRUZIONI PER LA GENERAZIONE

1. **Formato**: Utilizzare un generatore di immagini AI (Midjourney, DALL-E, Stable Diffusion, ecc.)
2. **Risoluzione**: Minimo 1024x1536 (proporzione verticale portrait)
3. **Salvataggio**: Rinominare ogni file secondo lo schema `{id}_{nome}.png`
4. **Posizionamento**: Salvare tutte le immagini nella cartella `images/characters/`
5. **Verifica**: Dopo aver generato le immagini, eseguire `python populate_all_characters.py` per popolare il database
